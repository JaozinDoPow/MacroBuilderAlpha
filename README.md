# MacroBuilder
  Description
MacroBuilder is a desktop application that allows users to create and execute macros (automated tasks and actions) without requiring programming knowledge.
Users can build sequences of actions and configure them to:
- Execute instantly
- Repeat in loops
- Run periodically at defined intervals

  Features
- Create macros using a visual interface
- Loop execution (instant repeat or timed interval)
- Save and load macro configuration
- Simple and beginner-friendly design
